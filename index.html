Začátek kariéry

Čemu se naučíte za 3 měsíce

1
Jak funguje internet
Důkladně se rozebereme, co je: HTTP, IP, DNS, hosting, doména, server, klient atd.
2
HTML a CSS
Naučíte se osvědčené nástroje (Flexbox, jmenování BEM, SASS) a s jejich pomocí se naučíte vytvářet rozvržení jakékoliv složitosti.
3
Responzivní rozložení
Naučíte se vytvářet projekty pro různé obrazovky a rozlišení: od mobilních zařížení až po širokoúhlé obrazovky, také vaše rozvržení bude vypadat perfektně ve všech prohlížečích.
4
Práce s návrhem
Naučte se pracovat s rozvržením pomocí praktického nástroje Figma a také se naučte pracovat s grafikou SVG, která se stala standardem ve web vývoji.
5
Workflow
Naučíte se používat editor kódu, pracovat v konzoli, naučíte se pracovat se systémem správy verzí Git a nahrát svůj projekt do repozitáře GitHub.
6
JavaScript
Budete mít ve svých rukou všechny základy: proměnné, funkce, objekty, pole, zpracování chyb a ladění.
